15. 交互规范

触控目标
最小点击区域 44×44px · 按钮高度不低于 44px · 按钮间距不低于 8px
弹出层显隐
必须使用 display:none/block 控制，禁止使用 transform 隐藏（避免事件穿透bug）
拍照流程
选中拍照项 → 快门可用 → 拍照（可多张）→ 合格/不合格激活 → 判定 → 锁定（不可追拍）→ 可"重新检查"撤销
判定项流程
选中判定项 → 快门隐藏 → 合格/不合格直接可用 → 判定 → 锁定
缩略条
只显示当前选中项的照片，切换选中项时缩略条跟随切换
已完成项
列表行显示 ✓ 或 ✗ 状态 · 选中时底部显示"↺ 重新检查"替代操作按钮 · 推荐下一项温和黄色脉冲高亮
